# SIPP: Sistem Informasi Pengelolaan Perpustakaan

SIPP (Sistem Informasi Pengelolaan Perpustakaan) adalah solusi perangkat lunak open-source yang komprehensif untuk mengelola operasional perpustakaan modern. Dibangun dengan teknologi ExtJS 3.3.1 untuk antarmuka pengguna yang kaya dan CodeIgniter sebagai framework PHP yang robust, SIPP menyediakan platform digital yang efisien dan user-friendly untuk mengotomasi proses manajemen perpustakaan.
